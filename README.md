# py101-4week

Учебный скрипт для отправки SMS-сообщений через сервис Twilio.
Выполнен в рамках курса Python 101 (неделя 4) — практика работы с внешними API
и переменными окружения.

## Возможности

- Отправка SMS-сообщения на произвольный номер через Twilio API
- Настройка учётных данных через переменные окружения (.env)
- Логирование результата отправки и обработка ошибок Twilio

## Установка

1. Клонируйте репозиторий и перейдите в его директорию.
2. Установите зависимости:

   pip install -r requirements.txt

3. Скопируйте `.env.example` в `.env` и заполните своими данными из личного кабинета Twilio:

   cp .env.example .env

## Использование

    python send_sms.py +79991234567 "Текст сообщения"

Если текст не указан, будет отправлено сообщение по умолчанию ("Привет!").

## Требования

- Python 3.8+
- Аккаунт Twilio с активным номером для отправки SMS

## Лицензия

Учебный проект, распространяется без ограничений для образовательных целей.
